# AI Document Summarizer

An AI-powered web application that extracts and summarizes PDF documents using Gemini AI.

## Features
- PDF Upload
- AI Summary Generation
- Summary History
- Database Storage

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- Gemini API
- SQLite

## Setup
1. Clone the repository
2. Install requirements
3. Add API key
4. Run the application
